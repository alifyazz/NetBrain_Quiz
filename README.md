### Deskripsi Aplikasi

NetBrain Quiz adalah platform kuis interaktif berbasis jaringan lokal (LAN) yang dirancang untuk menguji pengetahuan pengguna secara real-time. Dikembangkan menggunakan Python dengan arsitektur client-server, aplikasi ini memanfaatkan protokol TCP untuk menjamin pengiriman data yang akurat dan sinkronisasi skor antar pemain tanpa hambatan.

## Fitur Utama
1. Multi-Client Sync: Mendukung banyak pemain sekaligus dalam satu sesi kuis menggunakan mekanisme multithreading.
2. Real-Time Scoring: Validasi jawaban otomatis di sisi server dengan pembaruan skor instan.
3. Dynamic Leaderboard: Papan peringkat interaktif yang menampilkan posisi pemain secara langsung setelah setiap soal.
4. Reliable Connection: Manajemen koneksi socket yang stabil untuk menangani proses join dan disconnect pemain.
